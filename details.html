Details

Objectives

The primary objective of the study is to develop a prototype that can match a humming input with the recorded music that exists in the database.

Artificial Nueral Networks will then be used as the major classifier for the music matching.

Conceptual Framework

The "Hum It!" application accepts humming voice as an input that will then be pre-processed through noise reduciton before having its features extracted.

The matching process with the help of the trained Artifical Neural Network and Dynamic Time Warping, takes place by comparing these features to the reference trained features database, allowing the application to ouput the top five matched songs.
